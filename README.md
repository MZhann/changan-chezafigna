# Чанган-бот 🚗

Телеграм-бот, который живёт в группе и отвечает по приколу, когда сообщение начинается со слова **чанган**.
Ответы генерит самая дешёвая модель OpenAI (`gpt-4o-mini`) — характер и контекст приколов лежат в [`src/persona.js`](src/persona.js).

## 1. Создать бота
1. Пишешь [@BotFather](https://t.me/BotFather) → `/newbot` → получаешь `BOT_TOKEN`.
2. **ВАЖНО для групп:** `/setprivacy` → выбираешь бота → **Disable**.
   Без этого бот в группе видит только команды и реплаи, а не обычные сообщения — и не отреагирует на «чанган ...».
3. Добавляешь бота в группу как участника.

## 2. Ключ OpenAI
Берёшь на https://platform.openai.com/api-keys. Закинь немного денег на баланс — `gpt-4o-mini` стоит копейки.

## 3. Локальный запуск
```bash
npm install
cp .env.example .env   # на Windows: copy .env.example .env
# впиши BOT_TOKEN и OPENAI_API_KEY в .env
npm start
```

## 4. Деплой на Railway
1. Залей репозиторий на GitHub.
2. На [railway.app](https://railway.app) → **New Project** → **Deploy from GitHub repo** → выбери репо.
3. Вкладка **Variables** → добавь:
   - `BOT_TOKEN`
   - `OPENAI_API_KEY`
   - (опционально) `OPENAI_MODEL`, `TRIGGER`
4. Railway сам соберёт через Nixpacks и запустит `npm start`. Всё.

> Бот работает на long polling — публичный URL и вебхуки не нужны. Встроенный health-сервер на `PORT` нужен только чтобы Railway не ругался на отсутствие открытого порта.

## Как настроить характер
Весь юмор и стиль — в [`src/persona.js`](src/persona.js). Меняешь `SYSTEM_PROMPT`, добавляешь свои примеры — и бот отвечает иначе.

## Примеры
- `чанган че за хуйня` → *я сам в ахуе*
- `чанган проедься по Илье` → *ща пусть с машины выйдет*
- `чанган где Илья` → *во мне. или хз, дома наверное*
